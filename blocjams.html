BLOCJAMS

Bloc Jams is a lightweight JavaScript music player application. It has a simple, streamlined interface that can serve as a backbone or starting point to more robust music applications. I used HTML, CSS, JavaScript and jQuery to build the elements such as play, pause, skip and volume control
